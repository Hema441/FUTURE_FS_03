# Aura Wellness Pro

Aura Wellness Pro is a premium full-stack MERN web application developed for a luxury holistic wellness clinic.

Built using MongoDB, Express.js, React, and Node.js, the platform provides a seamless digital experience for both clients and administrators. 

The application features an elegant and responsive landing page with smooth Framer Motion animations, a smart appointment booking system with real-time slot availability, a client self-service portal for managing appointments without login, and a secure admin dashboard for complete booking management and analytics.

Administrators can confirm, cancel, edit, reschedule, or permanently delete appointments while monitoring projected revenue and booking statistics.

The project follows a modern luxury-inspired UI design with a forest green, gold, and cream color palette, rounded layouts, glassmorphism-inspired modals, and responsive components optimized for all devices.

Aura Wellness Pro demonstrates full-stack CRUD operations, REST API integration, real-time booking validation, modern frontend architecture using React and Vite, and scalable backend development with Express and MongoDB.
